    class Matrix

    La clase Matrix hereda de la clase 'Matrix_base' y será la que se utilize para definir las matrices densas.

                        @matriz[i][j]==other.matriz[i][j]

end
    encontrar() { |val| ... } click to toggle source

    Este método recorre la matriz y devolverá la posición de la matriz si lo que se realiza en el bloque llamado por yield es verdadero, si lo que se realiza en el bloque no se cumple, devuelve nil.

    # File lib/mymatrix/matrix.rb, line 87
def encontrar
        @ancho.times do |i|
                @alto.times do |j|
                        val = @matriz[i][j]
                        return [i,j] if yield(val)
                end
        end
        return nil
end

def to_s
        tmp="["
        @ancho.times do |i|
                tmp+="["
                @alto.times do |j|
                        tmp+="#{@matriz[i][j]}"
                        if j < (@ancho-1)
                        tmp+=","
                        end 
                end
                tmp+="]"
                if (i< (@ancho-1))
                tmp+=","
                end  
        end
        tmp+="]"
        tmp
end
