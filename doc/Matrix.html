    class Matrix

    La clase Matrix hereda de la clase 'Matrix_base' y será la que se utilize para definir las matrices densas.

    El initialize realiza dos funciones:

    • Con el comando 'super' llama al initialize de la clase madre.

    • Otorgarle a la matriz el alto y el ancho deseado.

        super

        @matriz.map! {Array.new(alto)}                
end

def ==(other)
        @ancho.times do |i|
                @alto.times do |j|
                        @matriz[i][j]==other.matriz[i][j]
                end
        end
end

def max

    La salida de este método es la matriz pasada a string, de manera que pueda usarse en 'puts', comprobaciones, etc.

    # File lib/mymatrix/matrix.rb, line 87
def to_s
        tmp="["
        @ancho.times do |i|
                tmp+="["
                @alto.times do |j|
                        tmp+="#{@matriz[i][j]}"
                        if j < (@ancho-1)
                        tmp+=","
                        end 
                end
                tmp+="]"
                if (i< (@ancho-1))
                tmp+=","
                end  
        end
        tmp+="]"
        tmp
end
